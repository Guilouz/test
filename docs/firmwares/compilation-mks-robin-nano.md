# Compilation du firmware pour MKS Robin Nano V3.0/V3.1

- Connectez-vous en SSH et saisissez ces commandes (une à la fois) :
``` yaml
cd ~/klipper/
```
``` yaml
make menuconfig
```

- Sélectionnez ces paramètres :

  Déplacez-vous sur le menu avec les touches ++arrow-up++ et ++arrow-down++ de votre clavier et validez l’élément sélectionné avec la touche ++enter++.
